# 二狗 (Ergou)

> 记得你说过的每句话，办过的每件事。

一个隐私优先、有温度、越用越懂你的个人AI助手。

## 特点

- **像人一样记忆** — 选择性记忆、会模糊、会回忆、有自己的判断
- **毒舌损友** — 说话带刺但没恶意，干活特靠谱
- **隐私优先** — 数据本地存储，不上传云端

## 目标平台

- Android（当前开发中）
- iOS（计划中）
- HarmonyOS NEXT（计划中）

## 技术栈

- Kotlin + Jetpack Compose
- Material 3
- DeepSeek API（前期）→ 自研模型（后续）
- Room (SQLite) + DataStore
- Ktor Client
